# Paper

This directory is the public reader entry point for the manuscript release.

- `paper.pdf` is the canonical reader version for the GitHub release.
- `paper.docx` is the editable manuscript source for the same public release.

The current public release uses a manual Word-to-PDF export. If the Word file changes, re-export `paper.pdf` before cutting the GitHub release tag so the PDF and the editable manuscript stay aligned.

The LaTeX manuscript tree under `paper/` remains available in the repository as secondary provenance, but it is not the primary public reading surface for this release.
