# Archive Contents

This document maps the public archive layout to the main project components.

## Code

- `multitimescale_memory/`: controller, routing, benchmarks, reporting, and journal-evaluation code
- `scripts/`: paper-building, campaign, and bundle-construction utilities
- `tests/`: regression and evaluation tests

## Frozen Artifacts

- `artifacts/popqa_500/post_fix/`: frozen recurring-heavy PopQA result
- `artifacts/freshness_v1/`: bundled controlled-update benchmark result
- `artifacts/journal_*`: broader seeded, calibrated, cross-family, and calibration-sweep bundles
- `artifacts/public_freshqa_*`: public-track provenance and manual audit bundles

## Papers

- `paper/arxiv/`: named preprint source
- `paper/tmlr/`: anonymous submission source
- `paper/shared/`: shared sections, tables, figures, bibliography, and claims map

## Submission Packages

- `deliverables/arxiv_submission/`: arXiv source package
- `deliverables/tmlr_submission/`: anonymous TMLR source package
- `deliverables/arxiv_preprint.docx`: editable Word manuscript

## Supplementary Material

- `docs/`: benchmark protocol and reproducibility notes
- `paper/shared/claims_to_artifacts.md`: mapping from paper claims to exact artifact files
